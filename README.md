# Flight Passenger Prediction using Machine Learning

## Overview
This project predicts the number of flight passengers using machine learning.

## Method
- Dataset loaded from online source
- Used Linear Regression model
- Features: Year, Month
- Target: Passenger count

## Evaluation
Model performance is evaluated using Mean Squared Error (MSE).

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. View output

## Tools Used
- Python
- Pandas
- Scikit-learn

## Learning Outcome
- Learned regression models
- Understood MSE
- Applied basic AI/ML concepts
